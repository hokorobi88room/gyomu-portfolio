# カフェ・コモレビ — 店舗ホームページ(4ページ)

## 誰の・どんな困りごとを解決するか

「店名で検索されたときに、ちゃんとした顔がない」小規模店舗(カフェ・美容室・整体・教室)。SNSだけでは営業時間・メニュー・場所が伝わらず、**機会損失が起きている**。制作会社の見積り(30万円〜)は高すぎる、という層へ「必要十分で美しい4ページ」を最短で届ける。

## 構成(情報設計)

| ページ | 役割 |
|---|---|
| トップ | 世界観+こだわり3点+営業時間(最重要情報をヒーローに) |
| メニュー | カテゴリ別・価格・説明文(検索流入の受け皿) |
| 店舗案内 | 営業時間・アクセス・設備(**来店判断に必要な情報を表形式で網羅**: 電源/Wi-Fi/支払い方法/予約可否) |
| お知らせ | 日付+タグつきリスト(更新運用まで設計) |

## 実装の特徴

- 素のHTML/CSS(4ページ共通CSS 1ファイル)。**画像ファイルゼロ**でも成立する配色・テクスチャ設計(CSSグラデーション)→ 実案件では店舗写真に差し替えるだけ
- **構造化データ(schema.org CafeOrCoffeeShop)**: 営業時間・住所を検索エンジンに正しく伝える(ローカルSEO)
- スティッキーヘッダー・現在ページ表示(aria-current)・明朝体の和の世界観
- レスポンシブ(375px/768px/1280px)

## 実案件ではこう応用できます

1. **業種展開**: 美容室(スタイル集+予約導線)/整体(症状別メニュー+W1の売れる構成)/教室(A3のフォーム受付と接続)
2. **Googleビジネスプロフィール連携**: 地図・口コミ・営業時間の一元管理までサポート
3. **お知らせのCMS化**: 店主がスプレッドシートに書くだけでサイトに反映される仕組み(GAS連携)
