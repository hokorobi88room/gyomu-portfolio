# 「伝わる資料」講座LP — 高コンバージョン縦長LP(解説モードつき)

## 誰の・どんな困りごとを解決するか

広告・SNSからの流入を申込みに変える「売れる1ページ」が欲しいオンライン講座・スクール事業者。きれいなだけのLPは作れる人が多いが、**「なぜこの順番でこのセクションなのか」を説明できる制作者は少ない**。

## このLPの見どころ: 💡制作解説モード

右下の「💡 制作解説を表示」を押すと、**全11セクションに設計意図の注記**が現れる:

> ①ファーストビュー(3秒で誰向け・何が変わるか)→ ②問題提起(あるあるで自分ごと化)→ ③共感(あなたのせいではない)→ ④解決策(3ステップの因果)→ ⑤カリキュラム(買った後の自分を想像させる)→ ⑥社会的証明 → ⑦権威性 → ⑧リスク反転(保証)→ ⑨価格アンカリング → ⑩FAQ(買わない理由の除去)→ ⑪クロージング

発注者はこの解説を読むだけで「構成を分かって作る制作者だ」と判断できる。**LPの発注前打ち合わせで、この画面を共有しながら構成を議論できる**のが実案件での使い方。

## 実装の特徴

- 素のHTML/CSS/JS 1ファイル(フレームワーク・画像ファイル不使用 → 高速表示)
- レスポンシブ(375px/768px/1280px)・スクロール連動フェードイン(IntersectionObserver・軽量)
- 締切カウントダウン・FAQアコーディオン(details/summary・JSなしで動作)
- OGP・メタディスクリプション設定済み

## 実案件ではこう応用できます

1. 商材(講座/サロン/物販/店舗集客)に合わせて**構成の型を組み替え**(BtoBなら「導入事例」を⑥に厚く等)
2. フォーム連携(A3のGAS受付自動化と接続し、申込→自動返信→台帳まで一気通貫)
3. ヒートマップ導線設計(CTA位置のABテスト前提のマークアップ)
