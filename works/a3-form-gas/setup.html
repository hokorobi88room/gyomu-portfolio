FormFlow 導入手順(所要 約20分)

必要なもの: Googleアカウントのみ(追加費用ゼロ)

1. フォームとスプレッドシートの用意

  1. Googleフォームを作成する。質問に「お名前」と「メールアドレス」を含める(名称は「氏名」「Email」等でも可)
  2. フォームの「回答」タブ → スプレッドシートにリンクする

2. 設定シート

リンクされたスプレッドシートに 設定 シートを作成し、A列にキー・B列に値:

A列 B列(例)
イベント名 夏休みプログラミング体験会
管理者メール you@example.com
確認メール本文 {{名前}} 様\n\n【{{イベント名}}】へのお申込みを受け付けました({{受付日時}})。\n当日の詳細は前日にご案内します。
定員 30(0 = 無制限)
Webhook URL (任意。Slack等のIncoming Webhook)

差し込みタグ: {{名前}} {{イベント名}} {{受付日時}}

3. スクリプトの設置

  1. スプレッドシートの 拡張機能 → Apps Script
  2. src/Code.gs の内容をエディタへ貼り付けて保存

4. トリガーの設定(重要: インストーラブルトリガー)

Apps Script左メニューの「トリガー」→「トリガーを追加」:

関数 イベントの種類
onFormSubmitHandler スプレッドシートから / フォーム送信時
sendDailySummary 時間主導型 / 日付ベース / 18〜19時(任意)

初回は権限承認ダイアログが出る(メール送信・フォーム操作の許可)。

5. 動作テスト

エディタで testWithDummySubmission を実行 → 自分宛てに確認メールが届き、台帳へ1行追加されればOK。その後、実フォームからも1件送信して確認する。

制限事項(正直に)

  • Gmailの送信上限: 無料アカウントは100通/日(Google Workspaceは1,500通/日)。大規模イベントは要相談
  • 確認メールのnoReplyは Workspace のみ有効(無料アカウントでは送信者のアドレスになる)
