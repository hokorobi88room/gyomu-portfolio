# InvoiceForge — Excel請求書自動発行システム(VBA)

## 誰の・どんな困りごとを解決するか

毎月、顧客リストと売上を見ながら請求書を1枚ずつ手作業で作成し、PDF保存してフォルダへ振り分けている個人事業主・小規模企業の経理担当。**「請求書の日」が半日仕事になっている**状態を解決する。

## Before / After

| | Before(手作業) | After(InvoiceForge) |
|---|---|---|
| 顧客20社の請求書作成 | 約4時間 | **約3分(ボタン1回)** |
| 消費税の端数・軽減税率 | 電卓で都度計算(ミス源) | 税率別に自動集計・端数処理は設定制 |
| 請求番号 | 前月のファイルを見て手で採番 | 発行履歴から自動採番(202606-001…) |
| ファイル整理 | 手動でフォルダ振り分け | 顧客別フォルダへ自動保存 |

## デモ（2通り）

1. ▶ **[ブラウザで動作を体験する](./demo/index.html)** — Excelが無くてもその場で動く。本体と同じロジック(税率別集計・端数処理3方式・源泉徴収10.21%・自動採番)をブラウザで再現。「一括発行」で5社分の請求書が数秒で生成される。
2. 📊 **実際に動くExcel体験版(.xlsm)** — 開いて「スタート」シートの緑ボタンを押すだけで、取引先5社分の請求書がまとまって出来上がる本物のマクロブック。作り方は [`build/組み立て手順.txt`](./build/組み立て手順.txt)（Windows+Excelで `build/請求書デモを組み立てる.vbs` をダブルクリックするだけ／手動でも3分）。サンプルは町工場（鉄工所）の部品加工請求を想定。

## 機能一覧

- 顧客マスタ×売上明細から対象月の請求書を全顧客分一括生成
- 適格請求書(インボイス)対応: 登録番号表示・税率別(10%/8%)の区分集計と端数処理
- 源泉徴収(10.21%)の控除欄(顧客ごとにON/OFF)
- PDF一括出力: `出力先/2026-06/C001_顧客名/202606-001_請求書.pdf`
- 発行履歴シートへの自動記録と月内連番の自動採番
- **実行前データ検査**: コード重複・日付不正・税率不正などを「検査結果」シートに一覧化し、問題ゼロになるまで発行しない(黙って壊れない)
- 設定シート方式: 自社情報・振込先・端数処理・支払期限ルールをコード変更なしで変更可能

```
sample-data/  顧客マスタ・売上明細のサンプルCSV
docs/SETUP.md 導入手順(15分)
demo/         ブラウザ体験デモ
```

## 動作環境

Windows版 Excel 2016以降(32bit/64bit両対応・外部ライブラリ参照設定不要)。

## 実案件ではこう応用できます

1. **見積書・納品書・領収書**への展開(同じ差し込み方式で3帳票セット化)
2. **販売管理ソフトのCSV**を売上明細として取り込み(弥生・freee等のエクスポートと接続)
3. **メール下書きまで自動生成**(Outlook連携で「請求書添付・宛先・件名」セット済みの下書きを作成)
