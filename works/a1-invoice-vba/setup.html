InvoiceForge 導入手順(所要 約15分)

対象: Windows版 Excel 2016以降(32bit/64bit両対応。Microsoft 365推奨)

1. ブックの準備

  1. 新規マクロ有効ブック(.xlsm)を作成し、次の7シートを用意する
  2. 設定 / 顧客マスタ / 売上明細 / 請求書ひな形(残り3シート=発行履歴・エラーログ・検査結果は初回実行時に自動作成)
  3. 顧客マスタ と 売上明細 に sample-data/ のCSVを貼り付ける(1行目=ヘッダー)
  4. 設定 シートのA列にキー・B列に値を入力する:
A列(キー) B列(値の例)
自社名 〇〇デザイン事務所
自社住所 東京都〇〇区…
自社TEL 03-0000-0000
振込先 〇〇銀行 〇〇支店 普通 0000000
登録番号 T0000000000000(適格請求書。なければ空欄)
対象月 2026/06/01
出力先フォルダ C:\Invoices
支払期限ルール 1(翌月末) or 2(翌々月10日)
端数処理 0(切り捨て)/1(四捨五入)/2(切り上げ)
源泉徴収欄 する / しない

2. 請求書ひな形シート

請求書ひな形 に請求書のレイアウトを作り、以下の「名前付きセル」を定義する(数式タブ→名前の管理):

宛先 宛先住所 請求番号 発行日 支払期限 自社情報 登録番号 振込先 明細開始(明細表の1行目左端。右へ 日付/内容/数量/単価/金額 の5列×20行) 小計10 消費税10 小計8 消費税8 源泉徴収 合計

印刷範囲をA4縦1枚に設定しておく(PDF出力にそのまま使われる)。

3. マクロの取り込み

  1. Alt+F11 でVBEを開く → ファイル→ファイルのインポート
  2. dist/cp932/ 内の6つの .bas を順にインポートする (注: src/ はUTF-8の正本。日本語Windowsには文字化け防止済みの dist/cp932/ を使う)
  3. マクロのセキュリティ警告が出る場合: ファイルのプロパティ→「許可する」にチェック(zip解凍後のブロック解除)

4. 実行

  • 一括発行: Alt+F8 → GenerateAllInvoices
  • 1社のみ: 顧客マスタで行を選択 → Alt+F8 → GenerateSelectedInvoice
  • 実行前に自動でデータ検査が走り、問題は 検査結果 シートに一覧される(問題ゼロまで発行しない設計)

5. トラブルシューティング

症状 対処
「シートが見つかりません」 シート名が §1 と完全一致しているか(前後の空白に注意)
「名前付きセルがない」エラー §2 の15個の名前がすべて定義されているか
PDFが出ない 出力先フォルダのパスが実在するドライブか。OneDrive同期中のフォルダは避ける
文字化け src/ ではなく dist/cp932/ をインポートしたか
