# ReportHub — 複数ブック自動集計ダッシュボード(Excel VBA)

## 誰の・どんな困りごとを解決するか

支店・担当者ごとの月次報告Excelが毎月20ファイル届き、**1つずつ開いてコピペで合算している**管理部門・経営者。しかも各支店の様式は微妙にバラバラ(列の並びが違う・ヘッダーの行位置が違う・「売上」だったり「金額」だったり)で、合算のたびに神経を使う。

## Before / After

| | Before(手作業) | After(ReportHub) |
|---|---|---|
| 20ファイルの合算 | 約8時間/月 | **約5分(ボタン1回)** |
| 様式のばらつき | 目視で読み替え(ミス源) | ヘッダー自動探索で吸収(列順・行位置・別名) |
| 経営レポート | 合算後にさらにグラフ作成 | 月次推移・前年比・担当者ランキングを自動生成 |
| 壊れたファイルの混入 | 気づかず合算 → 数字が合わない | 「取込結果」シートに成功/警告/失敗を明示 |

## 機能一覧

- 指定フォルダ内の全報告ブック(.xlsx/.xls)を自動走査・統合(**元ファイルは読み取り専用で開き一切変更しない**)
- **様式ゆらぎの吸収**: ヘッダー行を上から10行探索し、別名(売上/金額/売上高、商品/品目/商品名、日付/売上日/取引日 等)・列順の違い・前置行を自動で読み替え
- 小計行・メモ行の自動除外(除外件数も報告)
- ダッシュボード自動構築: 月次推移(前年同月比・マイナスは赤字強調)/担当者別ランキング/商品別TOP5(データバーつき)
- 取込結果レポート: ファイルごとに 成功(n行)/警告(ヘッダー不備)/失敗(破損等)を色分け一覧
- 統合データは一括書き込みで高速(セル1個ずつ書かない)

## 動作を試す(サンプルデータ同梱)

`sample-data/reports/` に**様式をわざとバラつかせた20支店分の報告ブック**を同梱(うち1冊はヘッダー不備で、警告検出の動作が確認できる)。SETUP後に「報告書フォルダ」をこのフォルダに向けて `RefreshDashboard` を実行するだけ。

## 構成・動作環境

```
src/          VBAソース(UTF-8正本・4モジュール)
dist/cp932/   日本語Windows向けインポート用
sample-data/  ダミー報告ブック20冊(tools/のスクリプトで再生成可)
docs/SETUP.md 導入手順
```

Windows版 Excel 2016以降。外部ライブラリ・参照設定不要。

## 実案件ではこう応用できます

1. **勤怠・経費・在庫報告**の集約(ヘッダー探索のキーワードを差し替えるだけ)
2. **メール添付で届く報告書**を自動でフォルダへ保存→集計まで(Outlook連携)
3. 統合データを **Power BI / スプレッドシート**へ連携する出口の追加

---
*※ サンプルデータの支店名・担当者名・数値はすべて架空です。*
